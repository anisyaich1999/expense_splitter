# 💸 Expense Splitter

A full-stack web app to split expenses between users and calculate balances automatically.

---

## 🚀 Live Demo
https://expense-splitter-zll6.onrender.com

---

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- HTML, CSS, JavaScript

---

## ✨ Features
- Add users
- Add expenses
- Automatic balance calculation
- View settlements
- Delete users and expenses

---

## 🚀 Deployment
- Backend: Render
- Database: MongoDB Atlas
- Frontend: Served from backend
